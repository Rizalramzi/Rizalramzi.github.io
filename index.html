About me

Hello! My name is Muhammad Rizal Ramzi, a student at SMK TELKOM SIDOARJO majoring in SIJA (Sistem Informasi Jaringan Aplikasi). I'm extremely enthusiastic about the world of technology and software development. My goal is to become a full stack developer with the ability to create holistic technological solutions from start to finish. Ever since I first entered the realm of computers, I've been fascinated by how software can transform the way we interact with the world. This fascination sparked my interest in software development. I have learned HTML, CSS, JavaScript, and PHP.

Currently, I'm focused on honing my skills as a full stack developer. I aim not only to have a strong understanding of how front-end and back-end work together but also to delve deeper into web and mobile application development. I believe that by comprehending the entire spectrum of software development, I'll be able to create more comprehensive and innovative technological solutions.
